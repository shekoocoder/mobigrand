Get Started

  • After Import the package into the scene where you will use go to the folder: Assets\UHUDText\Content\Prefabs\ 
  •  and drag into the scene the prefab 'HUDText'.
  • Now are ready for use, simple right.

 

NOTE: is necesary that only one bl_HUDText found in each scene, otherwise will not work as expected.

Constructor

When you want to show a new text, you need build the info for send it, for this is use the 'constructor'

HUDTextInfo, this script contain all variables for send the information to the text, you simple need create a new instance of this and fill the variable that you need, here I will explain for what is each variable:

 

HUDTextInfo
CacheTransform is the transform where the text will show.
Text the text for show in the UIText
Side the side from the text will out.
Size Size of UIText
Color Color of font
Speed Speed with the Text will out in horizontal sides
VerticalAceleration Speed with the Text will out in vertical sides
VerticalFactorScale Speed with the Text will decrease the scale
AnimationType Animation with the text will appear
VerticalPositionOffset Extra vertical position where the text will appear.
AnimationSpeed Animation speed
ExtraDelayTime Extra time before the text start to out
ExtraFloatSpeed Extra speed for float the text
TextPrefab Prefab for override the default text prefab
FadeSpeed Extra Speed for fade the text
